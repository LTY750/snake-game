# 📖 Introduction

This is a Snake game project built entirely with **native front-end technologies**.

In addition to preserving the classic Snake gameplay, the project also includes many modern enhancements focused on:

- Better interaction experience
- Rich visual presentation
- Mobile compatibility
- Complete project structure for showcasing

Compared to traditional “open-and-play” Snake games, this version is designed more like a polished showcase project suitable for:

- GitHub portfolio projects
- Canvas animation learning
- Front-end practice projects
- Beginner game development references

# ✨ Features

## 🎮 Multiple Game Modes

- 🏁 **Classic Mode**  
  Traditional Snake gameplay. The snake speeds up as you score higher.

- 🧱 **Maze Mode**  
  Obstacles are added to the map, requiring more careful movement.

- ⏱️ **Time Attack Mode**  
  Score as many points as possible within 60 seconds.

- 🌊 **Wrap Mode**  
  Walls are no longer deadly — the snake appears from the opposite side.

- 🏃 **Sprint Mode**  
  Faster starting speed and more intense gameplay.

# 🕹️ Gameplay Flow

## 1. Open the Website

You will see:

- The game canvas
- Score panel
- Mode selection cards

---

## 2. Choose a Mode

Each mode card includes a short gameplay description:

- Classic → Standard Snake gameplay
- Maze → Obstacles added
- Time Attack → 60-second challenge
- Wrap → Pass through walls
- Sprint → High-speed challenge

---

## 3. Start the Game

After clicking `START`:

- Mode cards will hide automatically
- The game begins
- Controls remain available

---

## 4. Game Over

After the game ends:

- Your score is displayed
- The result is saved locally
- You can switch modes and restart

# 🧠 Game Modes

## 🏁 Classic Mode

The standard Snake experience.

The snake speeds up every time food is eaten.
Colliding with walls or itself ends the game.

---

## 🧱 Maze Mode

Obstacles appear on the map.

Players must avoid both the snake body and the maze blocks.

---

## ⏱️ Time Attack Mode

Score as many points as possible within **60 seconds**.

The final 10 seconds trigger an intense countdown state.

---

## 🌊 Wrap Mode

The snake can travel through walls and appear on the opposite side.

The challenge becomes managing snake length and movement.

---

## 🏃 Sprint Mode

Higher initial speed and faster acceleration.

Designed for players who enjoy high-intensity gameplay.
